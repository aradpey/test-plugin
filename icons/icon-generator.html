AI Cover Letter Generator - Icon Generator

This page displays the icons for the browser extension. To create the actual PNG files:

Step 1: Right-click on each icon below and select "Save image as..." or take a screenshot
Step 2: Save them as:
  • icon16.png (16x16 pixels)
  • icon48.png (48x48 pixels)
  • icon128.png (128x128 pixels)
Step 3: Place the PNG files in the aicover-extension/icons/ directory
Alternative: Use any image editing software to create icons with the same design and dimensions
